# Camisetas — C.F.D.L.

T-shirt designs for the collective, delivered as PNG mockups. Each shirt is authored as a
self-contained HTML file under `src/` and rendered to PNG with headless Google Chrome, so the
mockups use the **real brand fonts** (FuturaStd, MinionPro) and the **site palette**.

All copy is lifted verbatim from the manifesto (`../docs/manifiesto.html`); titles are
sentence-case italic serif; em dashes, never hyphens. See `../CLAUDE.md` for the full voice.

## Garment & print

- **Garment:** natural / off-white tee (`#f1ece0`).
- **Inks:** Sapphire `#332f8a` (primary), Factory Yellow `#fde700` (accent only), Candy Pink
  `#f8ccce` (sparing). Two to three screens per design.

## Re-rendering

Edit the HTML in `src/` (shared styling + fonts + the tee silhouette live in `src/_shared.css`),
then re-render from the repo root:

```sh
CHROME="/Applications/Google Chrome.app/Contents/MacOS/Google Chrome"
for f in 01-brujula-sin-iman 02-marca-blanca 03-sello-marca-blanca 04-maleza; do
  "$CHROME" --headless=new --disable-gpu --hide-scrollbars \
    --force-device-scale-factor=2 --window-size=1600,2000 \
    --screenshot="$PWD/tshirts/$f.png" "file://$PWD/tshirts/src/$f.html"
done
```

Output is 3200×2000 px (2× scale). The contact sheet is regenerated by the Pillow snippet used
to build it (loads the four PNGs into a 2×2 grid, captions in MinionPro).

> Note: these are **flat mockups** showing the print on a garment silhouette — not
> production-separated artwork. For a printer, export each `.print` block at higher resolution
> or hand off the SVG/vector source per ink.
